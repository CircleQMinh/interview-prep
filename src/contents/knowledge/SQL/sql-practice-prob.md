# Practice questions you can use on this database

## Basic

1. Return all employees ordered by salary descending.
2. Find all customers from Vietnam.
3. Return all completed orders.

## Joins

4. Show employee name with department name.
5. Show order id, customer name, and order date.
6. Show order item details with product name.

## Aggregates

7. Count employees in each department.
8. Find average salary by department.
9. Find total number of orders per customer.

## HAVING

10. Find customers with at least 2 orders.
11. Find departments with more than 2 employees.

## Subqueries

12. Find employees earning above average salary.
13. Find products more expensive than the average product price.

## EXISTS / NOT EXISTS

14. Find customers who have placed at least one order.
15. Find customers who never placed an order.

## Ranking / window functions

16. Rank employees by salary within each department.
17. Find the highest-paid employee in each department.
18. Find the second highest salary in the company.

## Business-style problems

19. Find the first order date for each customer.
20. Find the latest order for each customer.
21. Find monthly revenue.
22. Find top 3 selling products by total quantity sold.
23. Find customers who bought `Laptop Pro`.
24. Find customers who bought `Laptop Pro` but never bought `Standing Desk`.
